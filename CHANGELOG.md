# Changelog

All notable changes to Chappe will be documented here.

This project follows a simple pre-1.0 changelog: user-facing changes, breaking
CLI changes, and security fixes should be listed under `Unreleased` until a
tagged release is cut.

## Unreleased

- Created standalone Chappe package and CLI.
- Added TDLib-backed auth flow with guided first-run onboarding.
- Added local SQLite analytics store for channels, posts, comments, drafts,
  policies, and audit events.
- Added channel sync, stats, graphs, post ranking, outlier detection, comment
  mining, ideas, briefing, and agent-context commands.
- Added policy-gated draft publishing with audit events.
- Added Codex, Claude Code, OpenCode, OpenClaw, and Hermes integration assets.
- Added Chappie mascot assets and public documentation.
