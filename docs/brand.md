# Chappe Brand Assets

## Naming

- Product: **Chappe**
- Mascot: **Chappie**
- Mascot role: **the little tower keeper**

Chappe is the CLI. Chappie is the tower keeper watching Telegram signals.

## Visual direction

Chappie should feel like a small open-source tool mascot: observant and
practical, calm by default. A little whimsy is fine; spam cues are not.

Core motifs:

- green hood
- wooden Chappe/semaphore tower
- signal arm
- binoculars/notebook/lantern/map/weather vane depending on feature
- soft cream background
- muted green palette
- simple outlines
- friendly but not childish

Avoid:

- robots
- sparkles
- paper planes
- generic chat bubbles
- growth charts as the main symbol
- Telegram logo imitation
